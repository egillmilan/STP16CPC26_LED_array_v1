# STP16CPC26 LED array v1
A 8x2 LED array controlled by a STP16CPC26 driver, meant for plugging into a custom board with a ÂµC for controlling the LEDs. Resistor R1 determines the constant current flowing through the LEDs, a 1k resistor results in 20mA flowing through each LED. The LEDs are high efficacy (~180 lm/W) 5630 LEDs from Samsung, part no. SPMWHT541ML5XAWMS5. The dimensions of the board are 21mm x 66.5mm with the pin header.
